jQuery Polaroid Gallery

html5 canvas + flash fallback

  • latest Update: 03/29/2012
  • By: FlashTang
  • codecanyon.net/user/FlashTang

jQuery HTML5 Polaroid Gallery is jQuery plugin ,if browser does not support HTML5 canvas tag ,will use flash version. e.g. if you open with IE8 .

HOW TO USE

1.Upload the PolaroidGallery to the same folder with your webpage (all necessary Javascript   icon image  css files are inside this folder).

2. Copy and paste below script to the head tag of your webpage :

<script src="PolaroidGallery/jquery-1.7.min.js"></script>
<script src="PolaroidGallery/library.js"></script>
<script src="PolaroidGallery/jquery.FlashTang.PolaroidGallery.js"></script>
<link rel="stylesheet" href="PolaroidGallery/style.css" />
<script>
         $(document).ready(function(){
              $("#demo-container").FTPolaroidGallery({
                          size:"900x558",
                          backgroundImage:"PolaroidGallery/bg.jpg",
                          rimWidth:4,
                          rim_radius:1,
                          rimBottomHeight:20,
                          cataThumbSize:"160x130",
                          cataThumbGrid:"2x1",
                          autoPlayDelay:6000,
                          fps:30,
                          BGM:"PolaroidGallery/music/bgm",
                          BGM_autoPlay:true,
                          BGM_volume:0.2,
                          fallBackData:"PolaroidGallery/fallback/gallery.xml"
                    });
             })
</script>

Parameters explain :

  size --------------------------> The div stage size of your gallery
  backgroundImage --------> Background image path
  rimWidth --------------------> The frame width of the drag-able photos
  rim_radius ------------------> The round radius photos (canvas only)
  rimBottomHeight --------->  The height of the title area of the photos
  cataThumbSize -----------> Catalogues thumb size
  cataThumbGrid------------> Catalogue chooser ,how may thumbs show in a row ,e.g. "2x1"
  autoPlayDelay -------------> The delay time of auto playing photos
  fps ----------------------------> Frame per second (canvas only)
  BGM -------------------------> Background music path (without expanded-name) .ogg .mp3
  BGM_autoPlay  -----------> Play background music at start. (on iOS always false)
  BGM_volume  -------------> The volume of background music
  fallBackData ---------------> The xml file for flash fallback

You should also eidt xml file for flash fallback  ,the demo xml is in floder : PolaroidGallery/fallback/gallery.xml

3. Create a div , (you can create this div inside a div which you want ) add galleries to it

    the div id should be same with $("#***********").FTPolaroidGallery ... 

    you can give it an anther id , e.g.  id="myPGallery"  ,  then use $("#myPGallery").FTPolaroidGallery ...
